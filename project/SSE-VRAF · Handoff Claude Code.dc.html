Instrucción para Claude Code
Implementa el SSE-VRAF como una aplicación Next.js 14 App Router con TypeScript, Tailwind CSS y Prisma + PostgreSQL. El prototipo navegable define el 100% del UX/UI — no redesignes ninguna pantalla. Cada sprint del prototipo corresponde a un módulo de la app. El modelo de datos ya está definido en el Data Studio. Los componentes del Form Builder se traducen directamente a componentes React reutilizables. El Process Builder define la lógica del motor de procesos. La automatización (SI→ENTONCES) se implementa como event handlers en el backend. Todo el sistema debe respetar el principio: capturar una sola vez la información para reutilizarla en todo el sistema.
